OpenAI To CNN

The AI project allows users to describe a cat or a dog with specific characteristics, such as color, breed, and size. Using OpenAI, we generate an image based on these characteristics and present it to the user. We then feed this image into a convolutional neural network to detect whether it is a cat or a dog, and inform the user of the result. Additionally, each generated image is saved to a database for future use in improving the model's accuracy. *** Under Construction ***
